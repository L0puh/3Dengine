# 3D ENGINE IN OPENGL
![](https://github.com/L0puh/3Dengine/blob/master/media/screenshot.png)
### current features:
- basic map editor (walls, ceiling, floor)
- loading models
- lighting
- basic collision handling
- text rendering 

### libs:
- opengl, glad, glfw
- glm for math
- imgui for debug windows
- stb image and assimp
